# 📊 Narrativa de Datos para Transformación Digital

## 📌 Descripción
Análisis del impacto de una transformación digital en una empresa, utilizando datos para generar insights estratégicos.

## 🎯 Objetivo
Traducir datos complejos en información clara para la toma de decisiones.

## 🛠️ Herramientas
- Análisis de datos
- Visualización
- Storytelling

## 📈 Resultados
- Mejora en productividad
- Identificación de riesgos organizacionales
- Recomendaciones estratégicas
